1. Mô tả đề tài: 
Mục tiêu chính của đề tài này chính là giúp mọi người tiếp cận, cũng như có thể cài đặt HUE qua các bước hướng dẫn chi tiết để có thể sử dụng được HUE trên môi trường docker windows, sau đấy nhóm sẽ thực hiện một demo có tên “Phần mềm quản lý quán trà sữa” được viết bằng ngôn ngữ Java, giao diện được thiết kế bằng JavaFX, sử dụng JDBC để kết nối với cơ sở dữ liệu MySQL trên HUE docker.
2. Hướng dẫn sử dụng:
Để có thể chạy được demo cần cài đặt MySQL 5.7 và HUE trên docker, để thuận tiện cho người sử dụng nhóm đã up sẵn file .tar container MySQL 5.7 đã tinh chỉnh theo link driver đính kèm (user: root, pass: my-secret-pw), CSDL dùng cho demo là db_milktea cũng được tích hợp sẵn trong container này, sau đấy người dùng cần cài đặt thêm HUE docker để có thể thực hiện việc kết nối từ project đến CSDL.
